# ابدأ هنا - دليل سريع 🚀

## خطوات الحصول على التطبيق (APK) بدون كمبيوتر:

### 1️⃣ سجل في GitHub (مجاني)
- روح: https://github.com/signup
- سجل حساب جديد

### 2️⃣ سوي Repository
- اضغط زر "+" فوق → "New repository"
- اسمه: `ScreenTranslator`
- خليه Public
- اضغط "Create repository"

### 3️⃣ ارفع الملفات
- حمّل وفك ضغط الملف الموجود فوق
- في صفحة الـ Repository: "Add file" → "Upload files"
- اسحب **كل** محتويات مجلد ScreenTranslator
- اضغط "Commit changes"

### 4️⃣ انتظر البناء (3-5 دقائق)
- روح تبويب "Actions"
- انتظر علامة ✓ الخضراء

### 5️⃣ حمّل APK
- في "Actions" → اضغط على "Build APK"
- تحت "Artifacts" → حمّل "app-debug"
- فك الضغط → احصل على `app-debug.apk`

### 6️⃣ ثبّت على جوالك
- انقل APK للجوال
- افتحه واسمح بالتثبيت
- خلاص! 🎉
