# طريقة بناء التطبيق باستخدام GitHub (مجاناً وبدون استهلاك بيانات كثيرة)

## الخطوات التفصيلية:

### الخطوة 1: إنشاء حساب GitHub (إذا ما عندك)
1. روح https://github.com/signup
2. سجل حساب جديد (مجاني)
3. تأكد من البريد الإلكتروني

---

### الخطوة 2: إنشاء Repository جديد
1. بعد تسجيل الدخول، اضغط على زر "+" في الزاوية اليمنى العليا
2. اختر "New repository"
3. سمّ الـ Repository: `ScreenTranslator`
4. خليه **Public** (عام)
5. **لا تضيف** README أو .gitignore أو LICENSE
6. اضغط "Create repository"

---

### الخطوة 3: رفع الملفات

**الطريقة الأسهل - استخدام واجهة GitHub:**

1. **حمّل الملف المضغوط الجديد** (اللي راح أعطيك إياه)
2. **فك الضغط** عنه
3. في صفحة الـ Repository الجديد، اضغط "Add file" → "Upload files"
4. **اسحب كل المجلدات والملفات** من مجلد `ScreenTranslator` إلى صفحة GitHub
   - تأكد إنك تسحب **كل شي** بما فيها المجلدات المخفية (`.github`)
5. في الأسفل، اكتب رسالة مثل: "Initial commit"
6. اضغط "Commit changes"

---

### الخطوة 4: بناء التطبيق تلقائياً

بمجرد رفع الملفات، GitHub راح يبدأ يبني التطبيق **أوتوماتيكياً**!

**لمتابعة البناء:**
1. روح إلى تبويب "Actions" في أعلى صفحة الـ Repository
2. راح تشوف عملية "Build APK" شغالة (دائرة صفراء تدور)
3. انتظر 3-5 دقائق حتى تتحول الدائرة إلى علامة ✓ خضراء

---

### الخطوة 5: تحميل التطبيق (APK)

بعد ما تخلص عملية البناء:

1. في صفحة "Actions"، اضغط على عملية "Build APK"
2. انزل للأسفل حتى تشوف قسم "Artifacts"
3. راح تلقى ملف اسمه **"app-debug"**
4. **اضغط عليه لتحميله** (حجمه تقريباً 10-15 ميجابايت فقط!)
5. فك الضغط عن الملف المحمّل
6. راح تحصل على ملف `app-debug.apk`

---

### الخطوة 6: تثبيت التطبيق على جوالك

**طريقة التثبيت:**

1. **انقل ملف `app-debug.apk`** إلى جوالك (عبر USB أو أي طريقة)
2. افتح الملف من جوالك
3. راح يطلب منك السماح بتثبيت تطبيقات من مصادر غير معروفة
4. اضغط "السماح" أو "Allow"
5. اضغط "تثبيت" أو "Install"
6. انتظر ثواني... وخلاص! 🎉

---

## ملاحظات مهمة:

✅ **البيانات المستهلكة:**
- رفع الملفات: ~3 ميجابايت
- تحميل APK: ~10-15 ميجابايت
- **المجموع: أقل من 20 ميجابايت فقط!**

✅ **مجاني 100%** - GitHub Actions مجاني للمشاريع العامة

✅ **كل مرة تعدل الكود وترفعه، راح يبني APK جديد أوتوماتيكياً**

---

## إذا واجهت أي مشكلة:

### المشكلة: عملية البناء فشلت (علامة ❌ حمراء)
**الحل:** 
- اضغط على اسم العملية الفاشلة
- شوف الأخطاء (Errors)
- عادة تكون بسبب ملفات ناقصة - تأكد إنك رفعت **كل الملفات**

### المشكلة: ما أقدر أثبت التطبيق على جوالي
**الحل:**
- روح إعدادات → الأمان
- فعّل "مصادر غير معروفة" أو "Unknown sources"
- حاول مرة ثانية
